== Description ==

SABoard is amazing community plugin writing, write comments, answers ... provide.

이 플러그인은 한국형 게시판 플러그인 입니다.
글쓰기,댓글쓰기,답변,비밀글 등의 기능을 제공합니다.
글쓰기 스킨,댓글 스킨, 글내용 seo적용 등등의 기능을 제공합니다.

== Installation ==

1. 플러그인활성화. (Plugin activate)
2. 게시판생성. (Board create)
3. 페이지에 숏코드 삽입. (Page in insert shortcode)
